Tofu.Auth Backend Service
=========================

This folder contains backend documentation for the Tofu.Auth service.

Contents
--------

- API and DTOs used to call the Auth service:
  - `Backend/Services/Tofu.Auth/API.md`
- User roles, tenant membership, permissions, and invitation flow:
  - `Backend/Services/Tofu.Auth/Roles_and_Tenants.md`

Local rules
-----------

When adding service-specific rules for Tofu.Auth (validation rules, DTO mapping notes,
error-handling conventions, etc.), add separate Markdown files in this folder and
link them from this `AGENTS.md`.
