# Комбинаторика — с нуля

> Назначение: личный конспект для изучения базовой комбинаторики (теория + задачи для самостоятельного решения).

Комбинаторика — это раздел математики о том, **как считать количество способов** что-то выбрать, расставить или скомбинировать, не перечисляя все варианты вручную.

---

## 0. Зачем и интуиция

Главный вопрос комбинаторики: «**Сколькими способами** можно сделать X?»

Прежде чем применять формулу, всегда задавай себе два вопроса:

1. **Важен ли порядок?** (AB и BA — это одно и то же или нет?)
2. **Можно ли повторять элементы?** (можно ли взять один и тот же элемент дважды?)

Ответы на эти два вопроса однозначно определяют, какую формулу использовать. Запомни это — это 80% успеха.

---

## 1. Два главных правила

Вся комбинаторика стоит на двух правилах.

### Правило произведения (И)

Если действие состоит из **нескольких шагов подряд**, и
- 1-й шаг можно сделать `n₁` способами,
- 2-й шаг — `n₂` способами,
- …
- `k`-й шаг — `nₖ` способами,

то всё действие целиком можно выполнить:

```
N = n₁ · n₂ · … · nₖ
```

**Ключевое слово — «И»**: нужно сделать и первое, и второе, и третье.

**Пример.** В меню 3 супа, 4 вторых блюда и 2 десерта. Сколько разных обедов «суп + второе + десерт»?
`3 · 4 · 2 = 24`.

### Правило суммы (ИЛИ)

Если объект можно выбрать из **нескольких непересекающихся групп**:
- из 1-й группы `n₁` способами,
- из 2-й — `n₂` способами,
- …

и выбирается **ровно один** из всех, то:

```
N = n₁ + n₂ + … + nₖ
```

**Ключевое слово — «ИЛИ»**: берём либо из первой группы, либо из второй.

**Пример.** На полке 5 книг по математике и 3 по физике. Сколькими способами взять **одну** книгу?
`5 + 3 = 8`.

> **Главное отличие:** «И» → умножаем, «ИЛИ» → складываем. Группы для сложения не должны пересекаться.

---

## 2. Факториал

`n!` («эн факториал») — произведение всех натуральных чисел от 1 до `n`:

```
n! = 1 · 2 · 3 · … · n
```

Примеры:
- `3! = 1·2·3 = 6`
- `5! = 1·2·3·4·5 = 120`
- **`0! = 1`** (по определению — это важно и пригодится в формулах)

Факториал растёт очень быстро: `10! = 3 628 800`.

---

## 3. Перестановки (порядок важен, берём ВСЕ элементы)

**Перестановка** — это способ расставить `n` различных элементов в ряд. Используются все элементы, важен порядок.

```
Pₙ = n!
```

**Почему:** на 1-е место можно поставить любой из `n` элементов, на 2-е — любой из оставшихся `n−1`, и так далее. По правилу произведения: `n · (n−1) · … · 1 = n!`.

**Пример.** Сколькими способами 4 человека могут встать в очередь?
`P₄ = 4! = 24`.

---

## 4. Размещения (порядок важен, берём НЕ все)

**Размещение** — выбираем `k` элементов из `n` и расставляем их по порядку. Порядок важен, повторов нет, но берём не все, а только `k`.

```
        n!
Aₙᵏ = ────────
       (n−k)!
```

**Пример.** В классе 10 учеников. Нужно выбрать старосту, его заместителя и казначея (3 разные должности). Сколькими способами?

Порядок важен (староста ≠ заместитель), берём 3 из 10:
`A₁₀³ = 10! / 7! = 10·9·8 = 720`.

---

## 5. Сочетания (порядок НЕ важен, берём НЕ все)

**Сочетание** — выбираем `k` элементов из `n`, **порядок не важен** (просто группа, набор). Повторов нет.

```
          n!
Cₙᵏ = ───────────
       k!·(n−k)!
```

Читается «це из эн по ка». Другое обозначение: `C(n,k)` или `(n над k)`.

**Связь с размещениями:** `Cₙᵏ = Aₙᵏ / k!` — сочетание это размещение, «поделённое» на число перестановок внутри группы (потому что порядок нам не важен).

**Пример.** В классе 10 учеников. Нужно выбрать 3 дежурных (должности одинаковые). Сколькими способами?

Порядок не важен (трое дежурных — просто группа):
`C₁₀³ = 10! / (3!·7!) = (10·9·8)/(3·2·1) = 720/6 = 120`.

> **Сравни примеры 4 и 5:** те же «3 из 10», но в размещениях должности **разные** (порядок важен) → 720, а в сочетаниях дежурные **одинаковые** (порядок не важен) → 120. Ровно в `3! = 6` раз меньше.

### Полезные свойства сочетаний

- `Cₙ⁰ = 1` (выбрать 0 элементов — один способ: ничего не взять)
- `Cₙⁿ = 1` (взять все — один способ)
- `Cₙ¹ = n`
- `Cₙᵏ = Cₙⁿ⁻ᵏ` (симметрия: выбрать `k` «что взять» = выбрать `n−k` «что оставить»)

---

## 6. Шпаргалка: какую формулу выбрать

| Порядок важен? | Повторы? | Берём | Формула | Название |
|---|---|---|---|---|
| Да | Нет | все `n` | `n!` | Перестановки |
| Да | Нет | `k` из `n` | `n!/(n−k)!` | Размещения |
| Нет | Нет | `k` из `n` | `n!/(k!·(n−k)!)` | Сочетания |

Сначала отвечаешь на «важен ли порядок?» и «берём все или часть?» — потом смотришь в таблицу.

---

## 7. Маленький бонус — размещения/сочетания с повторениями

Пока только для ознакомления, на первом этапе можно пропустить.

- **Размещения с повторениями** (порядок важен, элементы можно повторять): `nᵏ`.
  Пример: PIN-код из 4 цифр — `10⁴ = 10000` вариантов (каждая из 4 позиций — любая из 10 цифр).
- **Сочетания с повторениями** (порядок не важен, можно повторять): `Cₙ₊ₖ₋₁ᵏ`.
  Пример: купить 3 мороженых из 5 сортов (сорта могут повторяться).

## 8. Задачи для самостоятельного решения

Решай по порядку. Для каждой задачи сначала ответь на вопросы «важен ли порядок?» и «берём все или часть?», потом выбери формулу. Ответы — в самом конце файла, не подглядывай раньше времени.

**Правило произведения и суммы**

1. В кафе 4 вида кофе и 3 вида пирожных. Сколькими способами заказать «кофе + пирожное»?
2. Из города A в город B ведут 3 дороги, из B в C — 4 дороги. Сколькими способами доехать из A в C через B?
3. На завтрак можно взять **либо** один из 3 видов каши, **либо** один из 5 видов бутербродов. Сколько вариантов завтрака (одно блюдо)?
4. Сколько существует трёхзначных чисел, у которых все цифры чётные? (Подсказка: подумай про первую цифру отдельно — она не может быть 0.)

**Перестановки**

5. Сколькими способами можно расставить 5 разных книг на полке?
6. Сколькими способами 6 человек могут сесть за круглый стол? (это перестановки в ряд — `6!`; про «круглый стол по-настоящему» подумаешь позже)
7. Сколько различных «слов» (любых перестановок букв) можно составить из букв слова `КОТ`?

**Размещения**

8. В соревновании 8 участников. Сколькими способами могут распределиться медали — золото, серебро, бронза?
9. Сколько двузначных чисел можно составить из цифр 1, 2, 3, 4, 5, если цифры в числе **не повторяются**?

**Сочетания**

10. В команде 12 человек. Сколькими способами выбрать 5 из них для поездки?
11. Сколькими способами из колоды в 36 карт можно выбрать 2 карты (порядок не важен)?
12. В группе 7 девушек и 5 юношей. Сколькими способами выбрать делегацию из 2 девушек и 2 юношей? (Подсказка: выбери девушек, выбери юношей, перемножь.)

**Смешанные (сначала определи тип!)**

13. В классе 15 учеников. Нужно выбрать 3 дежурных. Сколькими способами?
14. В классе 15 учеников. Нужно выбрать старосту и его заместителя. Сколькими способами?
15. Сколько диагоналей у выпуклого восьмиугольника? (Подсказка: диагональ — это пара вершин, но из всех пар надо вычесть стороны.)
